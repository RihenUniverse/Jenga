# 26_xbox_uwp_dev_mode

Demonstration du flux Xbox UWP Dev Mode explicite avec les 4 types:
- `UwpStaticMath` (`staticlib`)
- `UwpSharedMath` (`sharedlib`)
- `UwpConsoleApp` (`consoleapp`)
- `UwpWindowedApp` (`windowedapp`)

## Build

```bat
cd /d E:\Projets\MacShared\Projets\Jenga\Jenga\Exemples\26_xbox_uwp_dev_mode
python ..\..\jenga.py build --platform XboxSeries-x64
```
