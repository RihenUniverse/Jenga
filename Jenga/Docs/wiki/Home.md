# Jenga Wiki

Bienvenue sur le wiki de Jenga.

Ce wiki est conçu pour te guider rapidement du niveau débutant au niveau avancé, avec des exemples concrets et utilisables directement.

## Démarrage rapide

1. Crée un workspace:

```bash
jenga workspace MonWorkspace
```

2. Crée un projet:

```bash
jenga project MonApp --kind console --lang C++
```

3. Build et run:

```bash
jenga build --config Debug
jenga run MonApp
```
